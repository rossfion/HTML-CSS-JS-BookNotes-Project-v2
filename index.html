Welcome to BookNotes 3.0

This is yet another reworking of a Node.js-based project I built for Udemy Instructor Dr. Angela Yu's Web Developer Bootcamp. It is inspired by David Siver's own website devoted to documenting all the non-fiction books he has read with some information about them. The covers are courtesy of the Open Library Book Covers API. This third version is built with HTML5, CSS Flexbox, CSS Grid, and Vanilla JavaScript. No frameworks for styling or JavaScript!

The previous version can be found here. It is version one because it a re-imagining of the project using front-end technologies only.
